Practice github and git

- git init - git status ( check if project have changes) - git add . (add all the changes in project) / git rm -- cached "filename" (remove changes in file) - git commit (commit the changes) - git push (push the changes and save to repository in github) - short cut "git add . && git commit -m "first commit" && git push" - git pull (to pull other changes in project that you didnt have, like your co-worker newly add in repository)
